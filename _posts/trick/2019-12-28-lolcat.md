Parliamo spesso di `lolcat`, ma che è?  
Serve semplicemente a *colorare l'output*!  
Installatelo con il vostro packkage manager preferito e datelo in pipeline ad un comando così:

```bash
comando | lolcat
```

Ad esempio si può usare con `echo`: 

```bash
echo "ciao" | lolcat
```

Oppure con `neofetch`:

```bash
neofetch | lolcat
```
